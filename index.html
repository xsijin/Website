Web Project #1: Moosage

Visit Website Visit GitHub See Demo

Project Description: My capstone project for the software engineering bootcamp at General Assembly. Moosage is a full-stack application that allows users to share moods and messages (moosages) on personal boards. Designed to emulate X/Twitter, this social media platform offers a seamless experience for user sharing.

Features: Users can create boards for various purposes such as birthday wishes, personal updates, blogs, and more. Boards can be set to public, allowing anyone to view them while restricting posts to logged-in users, or private, where only the user can view and post moosages. This flexibility allows users to tailor the Moosage experience to their specific needs.

Technologies Used: MERN stack (MongoDB, Express.js, React.js, Node.js), Tailwind CSS/DaisyUI, JavaScript, CSS, HTML

Security: JWT authentication with a 72-hour timeout, passwords hashed using CryptoJS before being stored in database.

Web Project #2: Dishcover

Visit Website Visit GitHub

Project Description: This project was a collaborative effort during my bootcamp, where we developed a full-stack platform for sharing food recipes and reviews. Our team of three was assigned specific roles, and I took the lead on frontend development. Additionally, I was responsible for developing the reviews model.

Features: Public users can search for recipes using a search bar or navigate through the Splide carousel. Members have the ability to create new recipes, upload photos of their dishes, and provide reviews for the recipes they've tried. The star ratings display the total number of reviews and the average rating for each recipe.

Technologies Used: MERN stack (MongoDB, Express.js, React.js, Node.js), Tailwind CSS/DaisyUI, Splide, JavaScript, CSS, HTML, RegEx (Regular Expression) for search bar

Security: JWT authentication with a 24-hour timeout, passwords hashed using CryptoJS before being stored in database.
